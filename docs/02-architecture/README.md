# Этап 2. Архитектурное проектирование

## Цель этапа

Определить архитектуру Movie Collection, распределить ответственность между Android-клиентом, Spring Boot backend и PostgreSQL, описать интерфейсы взаимодействия и зафиксировать ключевые архитектурные решения.
