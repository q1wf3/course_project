# Расширенный глоссарий

| Термин | Подробное описание |
| --- | --- |
| Авторизация | Проверка JWT-токена при обращении к защищенным endpoints. |
| Аутентификация | Вход по email и паролю с выдачей JWT. |
| Карточка фильма | Набор данных фильма, отображаемый в Android-приложении. |
| Оффлайн-режим | Возможность открыть ранее загруженную коллекцию из Room без доступного backend. |
| Админка | Экран администратора для просмотра пользователей, ролей, статистики и коллекций. |
| Docker Compose | Способ запуска backend, PostgreSQL и pgAdmin как связанных сервисов. |
| JaCoCo | Инструмент измерения покрытия backend unit-тестами. |
