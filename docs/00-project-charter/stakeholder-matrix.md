# Матрица стейкхолдеров

| Стейкхолдер | Интерес | Влияние | Ожидание |
| --- | --- | --- | --- |
| Студент-разработчик | Выполнить курсовой проект и защитить приложение. | Высокое | Рабочий Android-клиент, backend, база данных, документация. |
| Руководитель проекта | Проверить соответствие методическим требованиям. | Высокое | Наличие требований, архитектуры, БД, тестов, отчета и демонстрации. |
| Пользователь USER | Удобно вести коллекцию фильмов. | Среднее | Быстрый поиск, сохранение данных, понятный интерфейс. |
| Администратор ADMIN | Контролировать состояние системы. | Среднее | Статистика, список пользователей, управление ролями. |
| Система Docker/PostgreSQL | Обеспечить воспроизводимый запуск. | Среднее | Стабильный compose-сценарий и сохранение данных в volume. |
