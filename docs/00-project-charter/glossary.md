# Глоссарий

| Термин | Определение |
| --- | --- |
| Movie Collection | Мобильное приложение для ведения личной коллекции фильмов. |
| Фильм | Карточка аудиовизуального произведения с названием, годом выпуска, режиссером, категорией и обложкой. |
| Коллекция | Набор фильмов, принадлежащий конкретному пользователю. |
| Запись коллекции | Связь пользователя и фильма с персональными полями: статус, оценка, заметка, избранное. |
| USER | Обычный пользователь, который управляет собственной коллекцией. |
| ADMIN | Администратор, который просматривает пользователей, роли, статистику и коллекции. |
| JWT | Подписанный токен, подтверждающий личность пользователя при обращении к REST API. |
| PostgreSQL | Основная серверная база данных проекта. |
| Room | Локальная база данных Android-клиента для оффлайн-кэша коллекции. |
| Swagger/OpenAPI | Веб-документация REST API backend. |
