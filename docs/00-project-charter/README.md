# Этап 0. Инициация и бизнес-анализ

## Цель этапа

Сформировать первичное представление о проекте Movie Collection: описать бизнес-проблему, границы системы, заинтересованные стороны, основные бизнес-процессы, риски и экономическую целесообразность разработки.
