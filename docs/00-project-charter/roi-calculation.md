# ROI-расчет

Проект является учебным, поэтому расчет выполняется как условная оценка трудозатрат и эффекта автоматизации.

| Показатель | Значение |
| --- | --- |
| Оценка трудозатрат | 180 часов |
| Условная ставка | 350 руб./час |
| Условная стоимость разработки | 63 000 руб. |
| Экономия времени пользователя | 5 минут на один поиск/обновление |
| Основной эффект | Быстрое хранение и поиск фильмов в одном приложении |

Формула условного ROI:

```text
ROI = (полезный эффект - затраты) / затраты * 100%
```

Для учебного проекта показатель ROI используется не как финансовое обязательство, а как обоснование полезности разработки.
