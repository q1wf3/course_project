# SWOT-анализ

| Сторона | Описание |
| --- | --- |
| Strengths | Мобильный интерфейс, оффлайн-кэш, PostgreSQL, JWT, админка, разделение ролей. |
| Weaknesses | Обложки задаются URL, нет интеграции с внешним каталогом фильмов, часть данных вводится вручную. |
| Opportunities | Добавление рекомендаций, внешнего API фильмов, статистики просмотра, публичных подборок. |
| Threats | Недоступность backend, ошибки синхронизации, некорректная настройка Docker или базы данных. |

SWOT-анализ показывает, что проект уже закрывает основные учебные требования, но имеет понятные направления дальнейшего развития.
