# Руководство пользователя

## Регистрация

1. Открыть приложение.
2. Нажать `Регистрация`.
3. Ввести email и пароль.
4. Подтвердить создание аккаунта.

## Вход

1. Ввести email и пароль.
2. Нажать `Войти`.
3. После успешного входа обычный пользователь попадает в список фильмов, администратор - в панель администратора.

## Добавление фильма

1. На экране списка нажать кнопку `+`.
2. Заполнить название, год, режиссера, жанры и длительность.
3. Указать статус просмотра.
4. При необходимости добавить оценку и заметку.
5. Нажать `Сохранить`.

## Поиск и фильтрация

1. Ввести часть названия в строке поиска.
2. Выбрать фильтр по жанру или статусу.
3. Просмотреть найденные фильмы.

## Оффлайн-режим

Если интернет недоступен, приложение показывает коллекцию из локальной базы Room. Добавление, редактирование и удаление фильмов сохраняются локально и не теряются после закрытия приложения. Когда сервер снова доступен, отложенные изменения синхронизируются с backend.

Приложение также сохраняет текущую сессию пользователя, поэтому после перезапуска открывается последний доступный экран без повторного ввода логина.

## Смена темы

1. Открыть `Настройки`.
2. Включить или выключить `Темная тема`.
3. Выбранная тема сохраняется после закрытия приложения.

## Удаление фильма

1. Открыть карточку фильма.
2. Нажать `Удалить`.
3. Подтвердить действие.

## Работа администратора

1. Войти под учетной записью администратора.
2. На панели администратора просмотреть общую статистику: количество пользователей, фильмов и записей коллекций.
3. Открыть карточку пользователя, чтобы увидеть его коллекцию фильмов.
4. При необходимости изменить роль пользователя кнопкой `Сделать ADMIN` или `Сделать USER`.
5. При необходимости удалить пользователя с подтверждением действия.
6. Для работы со своей коллекцией нажать `Моя коллекция`.

Администратор не может удалить собственную учетную запись или снять с себя роль ADMIN.
