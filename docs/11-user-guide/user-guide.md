# Руководство пользователя

## Регистрация

1. Открыть приложение.
2. Нажать `Регистрация`.
3. Ввести email и пароль.
4. Подтвердить создание аккаунта.

## Вход

1. Ввести email и пароль.
2. Нажать `Войти`.
3. После успешного входа откроется список фильмов.

## Добавление фильма

1. На экране списка нажать кнопку `+`.
2. Заполнить название, год, режиссера, жанры и длительность.
3. Указать статус просмотра.
4. При необходимости добавить оценку и заметку.
5. Нажать `Сохранить`.

## Поиск и фильтрация

1. Ввести часть названия в строке поиска.
2. Выбрать фильтр по жанру или статусу.
3. Просмотреть найденные фильмы.

## Оффлайн-режим

Если интернет недоступен, приложение показывает последнюю сохраненную коллекцию из локального кэша. Изменения, требующие сервера, становятся доступны после восстановления соединения.

## Удаление фильма

1. Открыть карточку фильма.
2. Нажать `Удалить`.
3. Подтвердить действие.
