# Code smells

| Проблема | Риск | Решение |
| --- | --- | --- |
| Бизнес-логика в контроллере | Сложность тестирования и сопровождения | Логика вынесена в `AuthServiceImpl` и `MovieServiceImpl`. |
| Повторное преобразование entity в DTO | Дублирование | Используется `MovieMapper`. |
| Смешение UI и сетевого слоя | Трудно поддерживать Android-код | Retrofit и Room вынесены в отдельные компоненты. |
| Неочевидная админка | Роль есть, но пользы мало | Добавлена статистика, пользователи, роли и коллекции. |
