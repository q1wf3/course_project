# Этап 6. Тестирование

## Цель этапа

Показать, как проверяется корректность Movie Collection: описать план тестирования, уровни тестов, покрытие backend-кода и результат формирования HTML-отчета JaCoCo.

## Артефакты этапа

| Артефакт | Содержание | Статус |
|---|---|---|
| [Общее описание тестирования](testing.md) | Подход к unit, integration и UI-проверкам | Выполнено |
| [План тестирования](test-plan.md) | Сценарии проверки авторизации, коллекции, поиска, админки и API | Выполнено |
| [JaCoCo-отчет](jacoco-report.md) | Команды запуска, путь к HTML-отчету и показатели покрытия | Выполнено |

## Диаграммы и изображения

В разделе нет отдельных скриншотов. HTML-отчет покрытия формируется локально после выполнения Maven-команды и хранится в каталоге сборки `target`, который не добавляется в git.
