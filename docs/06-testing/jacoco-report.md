# JaCoCo-отчет

HTML-отчет формируется командой:

```bash
cd backend
mvn test jacoco:report
```

Локальный путь к отчету после генерации:

```text
backend/target/site/jacoco/index.html
```

Каталог `backend/target` является результатом сборки Maven и не должен добавляться в git. В репозитории хранится только описание команды и итоговые показатели покрытия.

Текущие показатели после расширения тестов:

| Метрика | Покрытие |
| --- | --- |
| Instructions | 53.22% |
| Branches | 50.00% |
| Lines | 60.05% |

Порог методических требований выше 40% выполнен.
