# План тестирования

| Область | Проверка | Инструмент |
| --- | --- | --- |
| JWT | Генерация, чтение и отказ при подмене подписи. | JUnit |
| Entity | Валидация фильма, статуса и оценки. | JUnit |
| Service | Создание фильма, поиск, вход, регистрация. | JUnit + Mockito |
| Controller | Делегирование запросов и проверка прав администратора. | JUnit + Mockito |
| Android UI | Вход, регистрация, создание фильма, поиск, настройки, профиль. | Ручное тестирование |
| Docker | Запуск `postgres` и `backend`. | Docker Compose |

Команда backend-проверки:

```bash
cd backend
mvn test jacoco:report
```
