# Спецификации методов

## `MovieServiceImpl.createMovie`

| Параметр | Описание |
| --- | --- |
| `userId` | Идентификатор владельца коллекции. |
| `command` | Данные фильма, статус, оценка и заметка. |

**Результат:** `MovieDto` созданной карточки.  
**Ошибки:** `IllegalArgumentException`, если пользователь не найден.

## `MovieServiceImpl.search`

| Параметр | Описание |
| --- | --- |
| `userId` | Идентификатор пользователя. |
| `query` | Часть названия фильма. Может быть пустой. |
| `status` | Необязательный фильтр по статусу просмотра. |

**Результат:** список `MovieDto`, отсортированный репозиторием по названию.

## `AuthServiceImpl.login`

| Параметр | Описание |
| --- | --- |
| `request.email` | Email пользователя. |
| `request.password` | Пароль в открытом виде, введенный на клиенте. |

**Результат:** `AuthResponse` с JWT, id, email и ролью.  
**Ошибки:** `IllegalArgumentException` при неверном email или пароле.

## `AdminController.changeRole`

| Параметр | Описание |
| --- | --- |
| `authorization` | JWT администратора. |
| `userId` | Пользователь, роль которого изменяется. |
| `role` | Новая роль. |

**Ограничение:** администратор не может снять роль `ADMIN` с самого себя.
