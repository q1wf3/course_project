# COCOMO

Оценка выполнена приближенно, потому что проект учебный и включает backend, Android-клиент и документацию.

| Показатель | Значение |
| --- | --- |
| Тип проекта | Organic |
| Оценочный объем | около 5 KLOC |
| Коэффициенты | a = 2.4, b = 1.05 |

Формула трудоемкости:

```text
PM = a * KLOC^b
```

Оценка:

```text
PM = 2.4 * 5^1.05 ≈ 13 человеко-месяцев
```

Для курсового проекта это значение используется как учебная модель. Фактическая разработка велась одним студентом и была ограничена сроками семестра.
