# Этап 12. Управление проектом

## Цель этапа

Показать планирование учебного проекта Movie Collection: декомпозицию работ, календарный график и оценку трудоемкости разработки.

## Артефакты этапа

| Артефакт | Содержание | Статус |
|---|---|---|
| [WBS](wbs.md) | Иерархическая структура работ по анализу, проектированию, реализации, тестированию и документации | Выполнено |
| [Диаграмма Ганта](gantt.md) | Календарный план выполнения этапов курсового проекта | Выполнено |
| [COCOMO](cocomo.md) | Учебная оценка трудоемкости и условной стоимости разработки | Выполнено |

## Диаграммы и изображения

В разделе используются PlantUML-диаграммы внутри файлов [WBS](wbs.md) и [диаграммы Ганта](gantt.md). При необходимости их можно экспортировать в PNG и вставить в пояснительную записку.
