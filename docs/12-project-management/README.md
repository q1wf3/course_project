# Этап 12. Управление проектом

## Цель этапа

Показать планирование учебного проекта Movie Collection: декомпозицию работ, календарный график и оценку трудоемкости разработки.

## Артефакты этапа

| Артефакт | Содержание | Статус |
|---|---|---|
| [WBS](wbs.md) | Иерархическая структура работ по анализу, проектированию, реализации, тестированию и документации | Выполнено |
| [Диаграмма Ганта](gantt.md) | Календарный план выполнения этапов курсового проекта | Выполнено |
| [COCOMO](cocomo.md) | Учебная оценка трудоемкости и условной стоимости разработки | Выполнено |

## Пояснение к разделу

WBS показывает состав работ проекта и помогает доказать, что разработка велась как полный цикл программной инженерии, а не только как написание кода. Диаграмма Ганта дополняет WBS календарным планом: этапы идут последовательно от инициации и требований к реализации, тестированию и подготовке отчета.

Оценка COCOMO используется как учебная расчетная модель трудоемкости. Она не заменяет фактический календарный план, но показывает примерную сложность проекта с учетом backend, Android-клиента, базы данных и документации.
