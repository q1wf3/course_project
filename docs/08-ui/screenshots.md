# Скриншоты и описание экранов

| Экран | Назначение |
| --- | --- |
| Вход | Авторизация существующего пользователя. |
| Регистрация | Создание новой учетной записи. |
| Главная | Просмотр коллекции, подборки и статистика. |
| Поиск | Быстрый поиск по названию и статусу. |
| Создание фильма | Добавление названия, года, режиссера, обложки, категории, статуса и оценки. |
| Детали фильма | Просмотр карточки и переход к редактированию. |
| Профиль | Имя, аватар, личные данные и переход к настройкам. |
| Настройки | Светлая/темная тема и параметры хранения. |
| Админка | Пользователи, роли, статистика, коллекции. |

Рекомендуемые скриншоты для отчета можно сохранить в `docs/images/`.
