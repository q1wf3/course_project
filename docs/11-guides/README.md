# Этап 11. Руководства

## Цель этапа

Подготовить эксплуатационные материалы для разных ролей: обычного пользователя, администратора и технического специалиста, который запускает или проверяет проект.

## Артефакты этапа

| Артефакт | Содержание | Статус |
|---|---|---|
| [Руководство пользователя](user-guide.md) | Вход, регистрация, добавление фильмов, поиск, профиль, настройки и оффлайн-режим | Выполнено |
| [Руководство администратора](admin-guide.md) | Работа с панелью администратора, учетными записями, ролями и коллекциями пользователей | Выполнено |
| [Техническая спецификация](technical-specification.md) | Технологии, структура проекта, запуск и проверка сервисов | Выполнено |

## Диаграммы и изображения

Отдельные изображения в разделе не хранятся. Пользовательские экраны и навигация описаны в [разделе интерфейса](../08-ui/README.md), а запуск сервисов - в [разделе развертывания](../10-deployment/README.md).
