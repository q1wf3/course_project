# Руководство администратора

## Вход

Начальная учетная запись администратора:

```text
admin@movie.local / admin123
```

После входа пользователь с ролью `ADMIN` попадает в панель администратора.

## Возможности админки

- просмотр количества пользователей;
- просмотр количества фильмов;
- просмотр количества записей коллекций;
- просмотр количества открытых жалоб;
- рассмотрение жалоб пользователей на карточки фильмов;
- перевод жалобы в статусы: новая, в работе, закрыта, отклонена;
- написать сообщение пользователю по жалобе;
- просмотр списка учетных записей;
- просмотр коллекции выбранного пользователя;
- изменение роли пользователя;
- удаление пользователя.

## Ограничения безопасности

Backend запрещает администратору:

- удалить самого себя;
- снять роль `ADMIN` с самого себя.

Административные endpoints находятся по пути:

```text
/api/admin/**
```

Для обращения к ним нужен JWT пользователя с ролью `ADMIN`.
