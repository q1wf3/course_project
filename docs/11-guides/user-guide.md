# Руководство пользователя

## Вход и регистрация

1. Откройте приложение Movie Collection.
2. Выберите вкладку входа или регистрации.
3. Введите email и пароль.
4. После успешного входа откроется главная коллекция.

## Добавление фильма

1. Нажмите кнопку `+` на нижней панели.
2. Заполните название, год выпуска, режиссера, URL обложки и категорию.
3. Выберите статус просмотра.
4. При необходимости поставьте оценку и добавьте заметку.
5. Нажмите сохранение.

## Поиск

Поиск доступен с нижней панели. Пользователь может искать фильм по названию и фильтровать коллекцию по статусу просмотра.

## Профиль и настройки

В профиле можно указать имя и аватар. В настройках доступно переключение светлой и темной темы.

## Оффлайн-режим

Если backend временно недоступен, приложение показывает последнюю загруженную коллекцию из локального Room-кэша.
